What is AkiraTV?

A Python-based local IPTV streaming platform

AkiraTV is a comprehensive streaming solution that transforms your video collection into a personalized TV experience. It supports multiple channel types including linear (24/7 scheduled programming), VOD (on-demand), and dynamic channels.

🎯 Key Features

  • Linear Channels: 24/7 scheduled programming with seamless transitions
  • VOD Channels: On-demand video playback with queue support
  • Dynamic Channels: Auto-generated content based on collections and tags
  • HLS Streaming: HTTP Live Streaming for broad compatibility
  • Hardware Transcoding: CUDA/NVENC support for efficient encoding
  • Web UI: Browser-based control interface
  • REST API: Full programmatic control via FastAPI
  • XMLTV Support: EPG generation for Kodi and other players

All workers inherit from BaseWorker which provides:

🔌 REST API

FastAPI-based HTTP interface

API Server

akiratv/api_server.py

Provides REST endpoints and WebSocket support for real-time updates.

POST /api/start
Response: {success: bool, message: string}
Start the AkiraTV engine
POST /api/stop
Response: {success: bool, message: string}
Stop the AkiraTV engine
POST /api/restart
Response: {success: bool, message: string}
Restart the engine
GET /api/status
Response: {running: bool, channels: int, ...}
Get current engine status
GET /api/channels
Response: ChannelStatus[]
List all channels with status
POST /api/channels/{name}/play
Body: {video_path: string}
Play video immediately on channel
WS /ws
WebSocket connection
Real-time status updates

Data Models

@dataclass class ChannelStatus: name: str type: str # "linear" | "vod" | "dynamic" enabled: bool status: str # "running" | "stopped" | "error" now_playing: str next_program: str viewers: int uptime: float @dataclass class LibraryStats: total_videos: int total_duration: float total_size: int resolutions: List[str] codecs: Dict[str, int]

Understanding the config.json structure

Default Configuration

{ "ffmpeg": { "hwaccel": "cuda", // Hardware acceleration "enable_subtitles": false, "transcoding": { "enabled": false, // Enable transcoding "bitrate": "auto", // auto | custom "custom_bitrate": "1500k", "video_quality": "source", // source | 1080p | 720p "encoder": "auto", // auto | h264_nvenc | libx264 "audio_quality": "copy", // copy | aac "fps": "auto", "threads": "2" } }, "storage": { "type": "disk", // ram | disk "ram_path": "./output", "disk_path": "./output" }, "output": { "mode": "http_hls", // http_hls | file "http": { "port": 8081, "bind": "0.0.0.0" }, "hls": { "segment_time": 6, // Seconds per segment "playlist_size": 4 // Segments in playlist } }, "channels": { "myAkiraTV": { "enabled": true, "type": "linear" // linear | vod | dynamic } } }
